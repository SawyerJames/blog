Vue.js文件目录介绍

Vue-cil是vue的脚手架工具：帮助我们编写好基础的代码的工具。 它能实现目录结构，本地测试，代码部署，热加载，单元测试等工作。

Vue-cil需要使用npm安装到全局环境。一般情况下，在国内npm安装速度过慢。有的时候还需要翻墙才能有点速度，所以这里推荐一款node的安装插件'snpm'，snpm是国内人士编写的一款直装并自动使用淘宝镜像的安装模式，使用简单，API易懂，详情戳这里

关于node,npm等安装及环境配置，本文不做过多介绍。

Vue的安装：先安装snpm，然后使用cmd写命令:

snpm install -g vue-cli

安装完毕后选择自己的包，按提示下载后完毕后使用命令:

 snpm install //安装package.json下的依赖

Vue目录文件介绍：

1：build与config，是webpack配置相关。

2：node_modules，npm install 安装依赖的代码库。

3：src，存放项目的源码，所有项目文件放在此处。

4：static，存放第三方资源。

5：.babelrc，是babel的一些配置。大部分浏览器不能直接支持es6语法，通常做法是把es6编译成es5。

5：.editorconfig，规定约束，属于编译器的设定。

6：.eslintgnore，忽视语法检查的目录文件。

7：.eslintrc.js，eslint的配置文件。
